# 학습내용
    1. 배열 기반 이진 최대 힙을 직접 구성하는 힙 정렬(Heap Sort) 로 O(n log n) 내림차순 정렬을 구현하고, 동점 처리(점수→구독개월→입력순서)로 정렬 안정성을 확보하는 방법을 학습했다.

    2. 정렬된 경계 배열에 대해 O(log k)로 점수 구간을 판정하는 이진 탐색(Binary Search) 을 적용해 채널을 4개 관심 그룹으로 분류하는 분할 정복 기법을 이해했다.

    3. OrderedDict로 get/set 평균 O(1)을 유지하고 용량 초과 시 최소 사용 항목을 축출하는 LRU 캐시를 구현하며, 해시 충돌·로드팩터·메모이제이션 개념을 파악했다.

    4. 경로 압축과 랭크 기반 합치기로 연산당 거의 O(α(n))을 달성하는 서로소 집합(Union-Find) 으로 유사 채널을 연결요소(취향 커뮤니티)로 묶는 그래프 군집화를 학습했다.

    5. 유사도 간선을 가중치 내림차순으로 정렬하고 사이클을 배제하며 강한 간선부터 채택하는 크루스칼 최대 신장 포레스트(MST) 의 그리디 선택 성질을 파악했다.

    6. 정규화 O(n), 정렬 O(n log n), 유사도 그래프 O(n²) 등 모든 핵심 연산에 점근 표기(O·Ω·Θ) 를 적용해 설계 단계에서 시간·공간 복잡도를 분석하는 방법을 학습했다.

    7. 수치 계산은 재현 가능한 결정론 알고리즘이, 자연어 해석·추천은 LLM이 담당하도록 역할을 분리해 시스템의 재현성과 검증성을 확보하는 설계 원칙을 이해했다.

    8. NP-hard에 가까운 채널 군집화를 유사도 임계값과 그리디로 다항 시간에 근사하고, 미적용 개념(동적 계획법·균형 트리)을 정직하게 표기하는 공학적 접근을 학습했다.
# 상세활동 내용
    결과물 : https://github.com/HANJUSON/Liberal_algorithm

    app.py의 _heapify로 배열 기반 이진 최대 힙을 직접 구성하고, 비교 키를 (점수, 구독개월, 입력순서) 튜플로 설계해 동점 채널도 결정론적으로 정렬되는 것을 코드로 확인했다.

    GROUP_BOUNDARIES = [20, 50, 80]경계 배열에 이진 탐색을 적용해 0~100 정규화 점수가 🌱/👋/😊/🔥 4그룹 중 어디에 속하는지 O(log k)로 판정하는 과정을 추적했다.

    OrderedDict 기반 LRU 캐시에 채널 ID + 활동·최근성 신호를 결합한 캐시 키를 넣어, 입력이 바뀌면 캐시가 자동 무효화되고 용량 초과 시 popitem(last=False) 로 축출되는 흐름을 검증했다.

    마지막 시청 후 경과일 d에 대해 score·e^(-0.005d) 지수 감쇠를 적용해, 오래된 채널 점수가 시간에 따라 줄어드는 Recency Decay 효과를 수치로 비교했다.

    채널을 카테고리 희소 벡터로 만든 뒤 코사인 유사도 ≥ 0.35 인 쌍만 간선으로 연결하고, Union-Find의 find에 경로 압축(parent[x]=parent[parent[x]])을 넣어 커뮤니티가 묶이는 과정을 확인했다.

    유사도 간선을 가중치 내림차순으로 정렬한 뒤 uf.union이 사이클을 만들면 버리고 강한 간선부터 채택해, 일반적인 최소 신장 트리와 반대인 최대 신장 포레스트가 구성되는 것을 비교했다.

    프론트엔드(static/app.js)의 drawTasteMap에서 반발력(∝1/d²) + MST 스프링 인력 + 중심 인력을 약 200회 반복하고 속도 감쇠(0.82)로 수렴시켜, 그래프를 SVG 취향 지도로 시각화했다.

    관심 키워드로 영상을 검색해 channeled빈도를 집계하고 2회 이상 등장한 채널만 후보로 채택하는 video-first 발굴로, 단발 오탐을 억제하며 추천 후보 풀을 구성했다.

    수치 분석(점수·정렬·그래프)은 결정론 알고리즘이, 추천·페르소나·정리 제안은 Upstage Solar LLM이 담당하도록 /api/analyze와 LLM 프록시 엔드포인트를 분리해 구현했다.
